# Методи API

## `getNextClaimTime(String playerName)`
Повертає час (у мілісекундах), коли гравець зможе отримати наступний кейс.

**Параметри:**
- `playerName` — ім'я гравця.

**Повертає:** `long` — час наступного отримання.

---

## `getClaimCooldown()`
Повертає стандартний кулдаун між відкриттями кейсів.

**Повертає:** `long` — тривалість кулдауну у мілісекундах.

---

## `getCaseName()`
Повертає назву кейса, який використовується у даній сесії API.

**Повертає:** `String` — назва кейса.

---

## `getKeysAmount()`
Повертає кількість ключів, доступних для відкриття кейса.

**Повертає:** `int` — кількість ключів.

---

## `isPending(String playerName)`
Перевіряє, чи має гравець невиконане отримання кейса.

**Параметри:**
- `playerName` — ім'я гравця.

**Повертає:** `boolean` — `true`, якщо кейс очікує отримання, `false` в іншому випадку.
